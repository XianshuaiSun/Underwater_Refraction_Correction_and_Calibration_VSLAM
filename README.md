# Multi-Layer Refraction Calibration and Correction Code

This repository is created to support the reproducibility of the study:

"An RGB-D SLAM Algorithm Based on a Multi-Layer Refraction Model for Underwater Scenarios"

---

## 📌 Repository Status

This repository has been established to provide public access to the materials associated with the above study.

The code for the proposed multi-layer refraction calibration and correction methods, as well as the simulation datasets and experimental recordings, are currently being finalized, organized, and documented.

All materials will be uploaded to this repository upon formal acceptance of the manuscript.

---

## 📦 Planned Contents

The repository will include:

- Implementation of the multi-layer refraction calibration algorithm  
- Implementation of the multi-layer refraction correction algorithm  
- Simulation datasets used for validation  
- Experimental underwater recordings
